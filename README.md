# My Calculator

A simple React calculator app built with Create React App.

## Features

- Basic arithmetic operations
- React-based UI
- Development server and production build support

## Getting Started

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm start
```

The app will be available at `http://localhost:3000`.

### Build for production

```bash
npm run build
```

## Project Structure

- `src/` - application source files
- `public/` - static public assets
- `package.json` - project metadata and scripts
- `.gitignore` - ignored files

## Notes

This project uses React 19 and `react-scripts` for build tooling.
